# Code of Conduct

Anthrimon adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

The full text is canonical at the link above. The summary:

- Be respectful. Disagreements are fine; insults are not.
- Assume good intent in others. Ask for clarification before assuming malice.
- The maintainers reserve the right to remove contributions, comments, or contributors who repeatedly violate the Covenant.

## Reporting

If someone's behaviour in an Anthrimon space (issues, PRs, discussions, chat) makes you uncomfortable, open a [private security advisory](https://github.com/purpledurpl075/Anthri-mon/security/advisories/new) on GitHub — that channel goes only to project maintainers, no public visibility. We'll respond within seven days.

Reporting is not retaliation — both reporter and reported get a fair read.

## Scope

This applies anywhere the project is represented: the GitHub org, mailing lists, conference talks, social media accounts using the project name, and any other space where someone might reasonably think they're interacting with the Anthrimon community.
